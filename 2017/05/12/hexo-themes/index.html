hexo Landscape themes 修改

在hexo themes可以找到很多別人提供的框架
點選進去透過git clone 到 自己的themes 資料夾下面
修改_config.yml 下面theme 為clone 下來的資料夾名稱
注意： 在theme: 後面要空一格 在填寫資料夾名稱

重新啟動server這樣就可以看到框架的改變
